{% trans "Interface de fácil acesso" %}

{% blocktrans %} Através do controle do teclado sem fio incluso no Efforia TV, ergonomicamente fabricado para funcionar como um gampepad, o Retro dispõe de uma interface de uso bastante prático, bastando selecionar o sistema de emulação que você deseja, e em seguida, o game que você quer jogar. Boa parte dos jogos são compatíveis com os emuladores. {% endblocktrans %}

{% trans "A solução para criação de jogos" %}

{% blocktrans %} Tendo à disposição diversas tecnologias multi-plataforma para gráficos, áudio e física, a criação de jogos para a plataforma torna-se bastante fácil e prática para os estúdios de jogos. Graças à tecnologia EOS-D somada ao nosso kit de programação, é possível conectar o Efforia TV ao notebook e utilizá-lo para testes e desenvolvimento. {% endblocktrans %}

{% trans "Instalação de jogos prática" %}

{% blocktrans %} Para a instalação de jogos novos no Retro, é muito fácil: basta acessar o ponto de rede pelo computador chamado RETRO e copiar as roms para a pasta respectiva. O software trata de procurar, catalogar e colocar a arte do jogo, bastando apenas selecioná-lo na interface para jogar. Para outros, é mais fácil ainda: basta instalar diretamente na Google Play Store. {% endblocktrans %}
